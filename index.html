Projects

Cyclistic Case Study

Cyclist-Case-Study is a data analysis project focusing on Chicago's Cyclistic bike-share program. Through the analysis of user behavior and trends, the project aims to drive strategic marketing initiatives. Key findings include seasonal ridership trends, differences between casual riders and annual members, and recommendations for increasing annual memberships. This project provides valuable insights for optimizing bike-sharing programs and promoting sustainable growth

Emotion Recognition

This project focuses on recognizing emotions from facial expressions using machine learning. We utilized the FER dataset, containing thousands of images depicting emotions such as happy, sad, and angry. Techniques like encoding, data augmentation, and dropout regularization were employed to train a convolutional neural network (CNN) model for accurate emotion classification.

Medical Insurance Cost Prediction

This project delves into medical insurance cost prediction, aiming to understand the factors influencing healthcare expenses. Leveraging the Medical Cost Personal Datasets, we explored variables like age, sex, BMI, smoking habits, and region to predict insurance charges. Techniques like linear regression, polynomial regression with regularization, and feature importance analysis were employed

Android App Market Analysis

The "Google Play Store apps and reviews" project analyzes over 10,000 Android apps, focusing on categories, ratings, sizes, prices, and user sentiments. It provides insights into dominant categories, user preferences for app size and price, and the correlation between app quality and pricing. This analysis aids developers and marketers in optimizing their apps for success on the Google Play Store.
